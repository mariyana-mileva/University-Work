What I Made

I tried to recreate the index page as closely as possible to the original, focusing on the navigation bar in particular because it was a layout I hadn't done before. For the first time, I worked with Flex and Grid, and I researched how to use them properly before applying this new knowledge when creating theÂ navigation bar.

Learning outcomes

During this task I was able to revise my knowledge of HTML and CSS as well as to learn new things like aligning items at the center of the navigation bar, work with videos in CSS and HTML and adding shadows to the elements. I also learned about Flex and gain more experience in working with Grid and how to create different layouts. At the end of the day I showed my work to the teacher and he was satisfied with the end result as I didn't have that much experience in HTML and CSS.

Working with Grid/Flex

Task of the assignment

We were given 4 website layouts that we were tasked to try and recreate using Grid/Flex. The challange here was to recreate them using only one HTML file and toggle between different CSS files using Javascript.

What I Made

I successfully recreated 3 out of the 4 examples given. I used only one HTML file and 3 CSS files which I toggle with buttons. In order to create the layouts I used display properties such as Grid and Flex which helped me position the elements on the page.

Learning outcomes

With this task, I learned how to use multiple styles on a single HTML page, as well as how to position elements on a page and learned new Flex functionalities such as Flex-directions and Gap. I also played around with different Gird layouts.

Json files and API

Task of the assignment

The task was to fetch and load information from a JSON file into our html page. We had to show our group members names and a picture of them. The second task was to try and load the information from an API. We had the freedom to choose which API we want to work with.

What I Made

For the first part of the assignment I created a simple html page displaying the names and pictures of my group members. I didn't focus on the page's style as my main priority was to make the fetching from the JSON file work. For the second task I chose to work with NumbersAPI (http://numbersapi.com/#42) This is an API that shows a random fact about the number that the user has entered.

Learning outcomes

With this task I learned how to fetch information from a JSON file as well as what an API is and how to work with it.

Clock

Task of the assignment

For this tasked we had to go through the JavaScript 30 website and choose a challenge that we want to try and create. I chose to do the clock.

What I Made

I created a simple analog clock, showing the current time. It has working hour, minute and second hands. I used the Date object to the current time. Afterwards by using transform properties I made the hands rotate according to the current hour, minutes and seconds.

Learning outcomes

This challenge was a nice exercise for me to use my knowledge in javascript as well as obtain new ones for the transform properties.
