Media production

Media production is the subject that teaches students how to create and produce various types of media content, such as audio, video, and digital media. We will learn how to develop and produce high-quality media content using production equipment, software tools, and editing techniques. We will also learn about media theory, ethics, and the effects of media on society.

Logo

Task of the assignment

The task was to create a logo for a brand of our choice (fictional, existing or our own) using Adobe Illustrator

What I Made

The task to design a logo was a bit of a challenge for me as I was not sure how to approach it. I ended up with the idea of combining the first letters of my first name and last name (Mariyana Mileva). I used black pink and white as for my color pallet as those are colors I like. I used linear gradient effect (From pink to white) on the letters and radial gradient (From black to darkpurple). For the creation of this logo I used Adobe Illustrator

Learning outcomes

With this task I get the opportunity to design my own personal logo for the first time. I learned the basics of working with Illustrator as well as various design tools and techniques. I learned how to export my work correctly in different formats (PNG, PDF, SVG). I learned about the different styles, for example: abstract logo, emblems, characters or just a text. Also learned what colors, fonts and images should I use to improve the designs.

Photography Portfolio

Task of the assignment

We had to borrow a DLSR camera from ISSD with which we had to take pictures that fit certan photography techniques and rules. We had to create a PDF Document created using InDesign that contains our best photos of each type. Some of the photos we had to edit externaly.

What I Made

I created a 43 page document containing photos of each type with explanations plus one bonus task. I started by creating the basic style for my portfolio. After that I slowly started adding the pictures for each task to it. For some of the images I used Adobe Lightroom and Photoshop to edit them in order to brighten or darken them. Some I edited with the goal of giving them different effect or feeling.

Learning outcomes

With this task I learned the basic techniques in the photography as well as how to edit my images externaly. I also learned how to work with the settings on the camera. How to apply different concepts such as Depth of Field, Waterfall effect, Picture composition, etc.

Short Movie

Task of the assignment

We had the task to create a short movie following the lectures for videography.

What I Made

Our team decided to create a fictional short movie about a homeless person who shares his story on how he ended up homeless. For this project I was responsible for recording the scenes as well as acting in one of them

Learning outcomes

With this assignment we had the opportunity to explore how are movies made, different types of frames and shots as well as learning the basics of video editing with DaVinci Resolve.
